**📝 To-Do List App**

A simple and efficient To-Do List application designed to help users manage daily tasks with ease. 🚀
Stay organized, track progress, and boost productivity with a clean and user-friendly interface.

**✨ Features**

**✅ Add Tasks**

Easily add new tasks to your list.

**🗑️ Delete Tasks**

Remove tasks once completed or no longer needed.

**✏️ Edit Tasks**

Update task details anytime.

**✔️ Mark as Complete**

Track completed tasks with a visual indicator.

**💾 Persistent Storage (Optional)**

Save tasks using local storage so they remain even after refreshing.

**📱 Responsive Design**

Works smoothly on desktop and mobile devices.
